# 🎨 Ankit Singh - Sticker Portfolio Website ✨

A super animated, vibrant sticker-themed portfolio website showcasing the skills, projects, and experience of Ankit Singh - an AI/ML enthusiast and Java developer.

## 🌟 Live Website Features

### ✨ Current Features
- **Interactive Hero Section** with animated stickers and glitch text effects
- **Particle System Background** with floating particles and dynamic connections
- **Sticker-Style Cards** with peel effects and hover animations
- **Animated Navigation** with smooth scrolling and mobile responsiveness
- **Skills Showcase** with interactive skill tags and color-coded categories
- **Project Gallery** featuring AI/ML and web development projects
- **Timeline Experience** showing professional internships with animated markers
- **Contact Cards** with social media integration
- **Mouse Follower Effect** with animated sticker cursor
- **Click Burst Effects** creating sparkles on user interactions
- **Scroll Reveal Animations** for smooth content appearance
- **Dynamic Background Gradients** that pulse and animate
- **Easter Egg** - Press Ctrl+Shift+R for emoji rain! 🎉

### 🎯 Technical Stack
- **HTML5** - Semantic structure with accessibility in mind
- **CSS3** - Advanced animations, gradients, and sticker aesthetics
- **Vanilla JavaScript** - Interactive features and particle effects
- **Font Awesome** - Social media and icon integration
- **Google Fonts** - Poppins and Fredoka font families
- **Canvas API** - Particle system rendering

- `stickerBounce` - Bouncing sticker effect
- `stickerPop` - Sticker appearance animation
- `stickerRotate` - Gentle rotation animation
- `floatGentle` - Floating movement
- `floatRandom` - Random floating paths
- `glitchAnimation` - Text glitch effect
- `miniStickerFloat` - Small sticker animations
- `skillTagPop` - Skill tag pop-in effect
- `iconBounce` - Icon bouncing
- `emojiWiggle` - Emoji wiggle effect
- `markerPulse` - Timeline marker pulse
- `heartBeat` - Heart icon beating
- `backgroundPulse` - Background pulsing

- **Mouse Follower** - Sticker follows cursor

- **Card Tilt** - 3D perspective tilt on hover
- **Scroll Reveal** - Elements fade in on scroll
- **Emoji Rain** - Secret Easter egg effect
- **Dynamic Gradients** - Animated background colors
- **Mini Stickers** - Sparkles on card hover
